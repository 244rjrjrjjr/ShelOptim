# ShelOptim

**Без исходников — только файлы для запуска.**

## Установка
Скачай **[ShelOptim-Setup.exe](https://github.com/244rjrjrjjr/ShelOptim/releases/latest/download/ShelOptim-Setup.exe)**  
Loader: папка → INSTALL → ярлык на рабочем столе.

## Файлы
- `ShelOptim-Setup.exe` — loader-установщик
- `ShelOptim.exe` + `ShelOptim*.dll` + `helpers` + `guide.html`
